Scss基础

2020-06-05 / 3 min read

一、Scss介绍

Sass 是一种 CSS 的预编译语言。它提供了 变量（variables）、嵌套（nested rules）、混合（mixins）、 函数（functions）等功能，并且完全兼容 CSS 语法。Scss 是 Sass 3 引入新的语法，其语法完全兼容 CSS3，并且继承了 Sass 的强大功能。

我们在写传统css的时候，经常会写一些重复度较高的代码。这是因为传统css文件缺少变量、函数等概念，但是现在使用Scss之后就可以通过新增的这些概念来达到提高效率的目的

二、基本使用

2.1 嵌套语法

在scss中，我们首先可以使用嵌套语法来写css了，这样可以极大提高文件的可读性，更加直观明了
eg：
css

.box {
   display: float;
}
.box ul {
    list-style: none;
}

Scss

.box {
    display: float;
    ul {
        list-style: none;
    }
}

2.2 父级选择器

scss提供了一个父选择器&，用于表示当前元素的父元素
eg:
css

.btn {
    background-color: #fff;
}
.btn  .active {
    background-color: #f00;
}

Scss

.bth {
    background-color: #fff;
    &.active {
        background-color: #f00;
    }
}

需要注意的是要在嵌套内部使用父级选择器，不然找不到父级元素会报错

2.3 嵌套属性规则

eg：
css

.box {
    font-family: fantasy;
    font-size: 1.2em;
    font-line-hight: 1.4;
    border-radius: 20px;
    border-color: blue;
}

Scss

.box {
    font: {
        family: fantasy;
        size: 1.2em;
        line-hight: 1.4;
    }
    border: {
        radius: 20px;
        color: blue;
    }
}

进阶使用

3.1 变量

scss中我们可以使用$声明变量

$width ：50px
.box {
    width: $width;
}

3.2 数学运算

.box {
    width: (100+20)*3px;
}

3.3 插值

我们可以使用插值#{}把变量包裹起来

$sizeKey ： 1
  .font-#{$sizeKey} {
    font-size: 12px
  }

p {
  $font-size: 12px;
  $line-height: 30px;
  font: #{$font-size}/#{$line-height};
    // font: 12px/30px;
}

3.4 导入文件

需要使用@import

@import 'style.scss';

3.5 媒体查询

// scss
$media: screen;
$feature: -webkit-min-device-pixel-ratio;
$value: 1.5;
.box{
    @media #{$media} and ($feature: $value) {
      height: 300px;
      width: 900px;
    }
}

3.6 继承

通过关键字@extend即可完成继承。

.box {
    height: 300px;
    width: 900px;
}
.smallBox {
    @extend .box;
    background: red;
}
// 效果为

.smallBox {
    background: red;
}
.box, .smallBox {
    height: 300px;
    width: 900px;
}

3.7 mixin和函数

使用@mixin指令声明一个函数，函数作用域内的属性都是return的值，使用@include使用函数

@mixin border-radius {
    -moz-border-radius: 5px;
    -webkit-border-radius: 5px;
    border-radius: 5px;
}
.box {
    @include border-radius；
}
// 无参数函数使用有点像继承

函数传参的时候我们还可以设置默认值

@mixin get-border-radius($border-radius: 5px){
    -moz-border-radius: $border-radius;
    -webkit-border-radius: $border-radius;
    border-radius: $border-radius;
}
.box1 {
    @include get-border-radius; 
    //不传参则默认为5px
}
.box2 {
    @include get-border-radius($border-radius:10px); 
    //结果为10px
}
